## Building

```bash
./gradlew build
```

The mod JAR will be in `build/libs/`

## Installation

1. Install Minecraft Forge 1.16.5
2. Copy `infinitygauntlet-X.X.X.jar` to `.minecraft/mods/`
3. Launch Minecraft with Forge
